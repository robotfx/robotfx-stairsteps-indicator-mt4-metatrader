The <strong>StairSteps indicator</strong> is a <strong>consolidation zone</strong> indicator who’s purpose is to draw the <strong>CZ</strong> on the chart and alert the trader when there are <strong>breakouts</strong> of this consolidation zones. This trading strategy does not rely on the trend as consolidation zones and breakouts occur regardless of the trending price.
